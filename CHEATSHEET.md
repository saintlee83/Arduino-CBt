# ⚡ 전자회로 핵심 치트시트

> 강의 슬라이드 + 복습강의 녹취 + Notion 총정리 통합 · **⭐ = 출제 예고** · 시험 직전 30초 컷

### 핀 선언 3종
- `int`(변경O·RAM) / `const int`(변경X·최적화·**강추**) / `#define`(텍스트치환·세미콜론X·타입X)
- 핀번호는 **const int**가 가장 안전

### PWM · analogWrite
- 핀 **3·5·6·9·10·11**(~표시) · 값 **0~255**(8비트) · **490Hz**(주기≈2ms)
- 우노 DAC 없음 → 진짜 아날로그 X (DAC는 Due). 실제 출력 0V/5V뿐, **밝기 ∝ 듀티사이클**
- `analogWrite(3,127)`≈절반 · 주파수 변경 = 타이머 **분주비(prescaler)**

### 풀업 / 풀다운
- 저항 없으면 **플로팅**(불확정). 외부 저항 **10kΩ** (SMD "103"=10×10³)
- **풀다운**: GND쪽·평상시 **LOW**·누르면 HIGH(상승)
- **풀업**: VCC쪽·평상시 **HIGH**·누르면 LOW(하강)
- 내부 풀업 ≈**20kΩ**(`INPUT_PULLUP`). 실무=내부풀업+하강감지. **외부풀다운+내부풀업 혼용 금지**
- 약한(>10k)=GND확인X / 강한(<10k)=눌러도 GND로 읽힘

### 디바운싱
- 채터링 = 접점 진동(수 ms). 택트스위치일수록 심함
- SW: 상태 변하면 **5ms 대기 후 재확인** → 같으면 인정. HW(RC)+SW 병행이 최선

### RGB LED
- 셀 3(R·G·B)+공통 1 = 4핀, 전류제한저항 3개
- 대부분 **공통 양극**(공통=VCC, 음극 제어) → 동작 반대: **0%=최대밝기, 100%=OFF**
- 셋 다 ON = 흰색

### ADC · analogRead
- **10비트 = 1024단계 = 0~1023** (0부터라 최대 1023)
- 기준 **5V**: 0V→0 · 2.5V→512 · 5V→1023 · 1스텝 = 5/1024 ≈ **4.88mV**
- N비트 → 2ᴺ (8→256 … 12→4096). 비트↑ = 미세변화 포착

### 센서 · 전압분배
- **Vout = Vin × R2 / (R1+R2)**
- 가변저항: 와이퍼가 R1:R2 비율 변경(전체저항 일정)
- CdS(광): 밝을때 **5kΩ** ↔ 어두울때 **200kΩ**. R1위치=밝으면↑ / R2위치=밝으면↓
- NTC 서미스터: 온도↑ → 저항↓

### map / constrain
- `analogRead` 0~1023(10비트) ↔ `analogWrite` 0~255(8비트) → **map 필요**
- `map(v,fLo,fHi,tLo,tHi)` 역방향(tLo>tHi) 가능 · `constrain(v,min,max)` 범위제한(overflow 방지)
- 자동조명: `map(val,200,900,255,0)` 역방향 → 어두울수록 밝게

### DC모터 · H-브리지
- 정지(다열림)/정방향(좌상+우하)/역방향(우상+좌하)/브레이크(같은단 단락)
- ⚠️ 같은쪽 위·아래 동시 ON = **쇼트**(금지). **방향 바꾸기 전 EN을 LOW로**
- **L293D**: 16핀·하프브리지4·**플라이백 다이오드 내장** / **L298**: 대용량·다이오드 외부필요·SENSE=전류판단
- EN에 PWM=속도. 데드밴드: **0~462 역 / 462~562 정지(중심512) / 562~1023 정**

### 전원 설계
- 전류한계: 데이터핀 **40mA** / USB **500mA** / 배럴잭 **1A**
- 모터전원 분리 이유: ①전류부하로 전압흔들림 ②브러시 노이즈 (**GND는 공통**)
- **7805**(L7805CV): 9V→5V, 입·출력 디커플링 커패시터(극성주의)

### 서보 모터
- 표준(0~180°, 펄스=**각도**) / 연속회전(펄스=**속도**)
- 펄스폭: **0°=1ms · 45°=1.25ms · 90°=1.5ms · 180°=2ms** · 주기 **20ms(50Hz)**
- `writeMicroseconds`가 `write`보다 정밀. 전원 분리(순간전류 큼), 9·10핀 PWM 제한
- 데드밴드("지잉" 떨림): 지터·부하·전압불안정·기어유격 → `detach()`·미세변화무시·별도전원(**2A+**)+평활 **100~1000µF**

### 스테핑 모터
- 브러시리스, 펄스당 1스텝. 단극(간단·토크½) / 쌍극(H브리지 필요·전체코일)
- **200스텝/회전 = 1.8°/스텝**(하프 0.9°), 8코일 2상, **12V**
- **NEMA-17 = 42×42mm**(크기 규격). 순서=방향, 속도=회전속도
- **탈조**: 너무 빠른 주파수 → 스텝 놓침. 라이브러리 사용 시 EN=HIGH

### 7세그먼트(FND) ⭐⭐ 출제예고
- LED 8개(a~g 7 + DP). **공통양극**(공통VCC·제어LOW) / **공통음극**(공통GND·제어HIGH=직관적)
- 4자리 다이나믹: **12핀**(개별이면 32핀)·잔상효과·330Ω
- 패턴(공통음극 ABCDEFG·DP, HIGH=ON): 0=`0xFC` 1=`0x60` 2=`0xDA` 3=`0xF2` 4=`0x66` 5=`0xB6` 6=`0xBE` 7=`0xE0` 8=`0xFE` 9=`0xF6`
- **F = a,e,f,g = 1000 1110 = `0x8E`** (애노드면 반전 `0x71`) · 2=abdeg=`0xDA` · A=abcefg=`0xEE`
- 요령: 0은 g·dp만 OFF · 1은 b,c만 · 8에서 빼서 문자 (F = E − d)
